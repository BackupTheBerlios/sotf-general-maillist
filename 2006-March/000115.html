hello,

I hope you had a good event and now I can add some more confusing thoughts 
to this issue:

I would stress to make a *conceptual* decision here, because we are talking 
about an open (package) format here, so arguments such as "it is easy to 
handle it under X or Y" should not be given high priority. It should be 
conceptually clear, and then it will be also easy to implement.

For example if you create both audio and video directories, then how do you 
know which is the real content? And if the main content is a SMIL 
presentation, then in which directory you would put it? It is of course 
possible to create a list of allowed directories and allow only one of them 
in a package...
