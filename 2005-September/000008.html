hi all,

we (Juergen, Markus and me) discussed the switch from CVS to SVN on  
berlios.de

it seems rather simple to convert the repository to SVN and we would  
really wanna make the switch because revisioning is simpler in SVN  
(each commit increases revision on the whole tree, making it easier  
to go a step back or to keep up with changes), branching is simpler  
(uses directories where you can simply copy and merge between  
branches and it's easier to read because no meta dimension for  
branches is used, just plain folders)

For the node admins this means:
- you have to backup your node installation
- delete code and www directory in your node
- checkout code and www directory from SVN
- copy config.inc.php from backup
- reapply modifications you did to either the files in code or www  
directory

Please comment on this issue.

Wolfgang

PS: please give quick response as we would really wanna switch soon ;-)
