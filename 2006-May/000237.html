An XBMF contains the info into which station/series it belongs.
The station data referred to in the XBMF cannot be found on the node.
This causes the problem.

For example, this is a station page on the node:
http://radio.sztaki.hu/node/showStation.php/094st3

in this case the station id is: 094st3
and you may put into XBMF either of these:
<stationid>3</stationid>
<stationid>094st3</stationid>

If you need further help, please send the XBMF metadata file (.xml).
