Hi Zoe,

thanks for contacting the StreamOnTheFly project. As you might have  
seen, its usage is intended for radio programme exchange. But it also  
fits perfectly for narrowcasting for specific groups or regions.

You can run StreamOnTheFly as a stand-alone node, but we certainly  
would encourage partners to join our metadata exchange network. We  
don't see any language problems on our side, we alreday have  
programmes with various languages in the node network. You simply  
might define an entry point for your site, which solely points to  
content produced by audiofoundation members. Remember, that the node  
user interface is made for journalists, not really for end users. But  
it is rather simple to integrate feeds to an existing portal, we have  
done that e.g. for PLONE (see http://www.achwelle.at/).

Best regards,

Roland
