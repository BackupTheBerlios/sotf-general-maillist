I forgot to add something useful:

Micsik András wrote:
> Hi all,
> 
> today I moved to SVN from CVS. It was simple:
>

first save your changes: go into the node dir and:
cvs diff >mychanges

> svn checkout 
> svn+ssh://YOURNAME@svn.berlios.de/svnroot/repos/sotf/node/trunk node.new
> 
> mv node node.old
> 
> mv node.uj node
> 
> mv node.old/repository/* node/repository
> 
> mv node.old/users/* node/users
> 
> cp node.old/www/config.inc.php node/www
> 
> find node -name CVS
> (delete found CVS directories)
> 
> ensure httpd has write access for
> node/www/tmp node/www/tmp/cache node/code/templates_c

now your node should be running OK, but it is 'out-of-the-box'

you have to add your local changes by hand again, based on the file 
mychanges

> and that's the end
> 
> good luck,
>         Andras
